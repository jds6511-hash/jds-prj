# 질의셋 초안 — 사람 확인 전 정식 평가에 사용 금지

이 문서는 `queries_draft.jsonl`의 검토용 사본입니다. 각 질의의 정답 구간(`gt_start`/`gt_end`/`gt_seg_idx`)은 세그먼트의 `subtitle`(Whisper 전사)·`caption`(VLM 장면 설명) 텍스트만 보고 추정한 것으로, 실제 영상을 재생하며 확인하지 않았습니다. 사람이 영상을 보고 구간·라벨을 확정하기 전까지는 평가에 사용하지 마세요.

**설계 메모**: 자막형(sp_q01~03)은 `subtitle` 필드(Whisper 전사, 세그먼트 idx0-1, 3, 22-23은 화면 밖 대사)를 근거로 삼았습니다. 복합형 중 sp_q07/sp_q08은 해당 세그먼트의 `subtitle`이 빈 문자열이라, `caption` 안에 VLM이 옮겨 적은 화면 속 대사("DNA 변이 억제법을 찾아내셨죠?", "뭐가 좋고 나쁜 건지 어떻게 판단하지?")를 대사 단서로, 나머지 장면 묘사를 시각 단서로 사용했습니다 — 통상적 의미의 `subtitle` 필드 대사는 아니므로 사람 검토 시 이 점을 확인해 주세요.

- **sp_q08**: 화염 장면이 idx 0, 1, 18에도 반복 등장해 시각 단서만으로는 변별력이 없고, 화면 속 대사로만 세그먼트를 특정한 상태 — 그 대사 자체도 caption의 자유 서술 안에 섞여 있어 정확히 옮겨졌는지 확신이 낮음.
